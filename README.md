# Portfolio

A minimal single-page portfolio inspired by [samwill.is](https://samwill.is/).

## Run locally

Open `index.html` in a browser, or use a simple server:

```bash
npx serve .
```

## Customize

- **index.html** — Update name, role, company, about text, case study links, and social links.
- **styles.css** — Adjust `:root` and `[data-theme="dark"]` for colors and fonts.
- Theme choice is saved in `localStorage` and respects system preference by default.
